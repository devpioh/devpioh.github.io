### 공통 단축키 설정
```
Alt + 1 : 정의로 이동(F12)
Alt + 2 : 뒤로 탐색

Shift + Alt + 1 : 다음 책갈피 이동
Shift + Alt + 2 : 이전 책갈피 이동
Shift + Alt + 3 : 책갈피 설정 해제
Shift + Alt + Ctrl + 3 : 모든 책갈피 제거
Shift + Alt + 4 : 책갈피 창 보기 

Ctrl + Shift + c : 주석 토글(VS에서는 ToggleComment 사용, Code에서는 그냥 사용)
```
