## 개요
로컬위치에서 원격지로 파일 전송
***
### scp

원격지로 전송
```
scp [옵션] [로컬 파일 경로] [원격지 계정]@[원격지 ip]:[전송 경로]

옵션
-r : 하위 디렉토리까지 전부 전송
-v : 과정 출력
-c : 압축
-a : 아카이브
-p : 원본 속성 유지
-P : 포트 번호 지정
```

로컬로 전송
```
scp [옵션] [원격지 계정]@[원격지 ip]:[원본 경로] [로컬 전송 경로]
```
